```

LAYOUT:

- Clean and professional
```

### Refinement Prompts

**Prompt 2: Theme Toggle Enhancement**
```
Add a fixed theme toggle button in the top-right corner that:
- Persists theme choice in localStorage
- Detects system preference on first visit
- Has smooth icon transitions
- Shows sun icon for light mode, moon for dark mode
- Has a subtle shadow for depth
```

**Prompt 3: Project Cards Improvement**
```
Make the project cards:
- Fully clickable (entire card, not just icon)
- Show hover effect with shadow and border color change
- Display GitHub icon in top-right
- Have smooth transitions
- Change title color to blue on hover
```

**Prompt 4: Scroll Indicator**
```
Add an animated scroll indicator at the bottom of the hero section:
- Bouncing arrow animation
- Smooth scroll to projects section when clicked
- Subtle gray color
- Positioned at bottom center
```

### 2. Theme Toggle Component

```typescript
// Generated theme toggle logic
const [isDark, setIsDark] = useState(false)

useEffect(() => {
  const savedTheme = localStorage.getItem("theme-preference")
  const prefersDark = window.matchMedia("(prefers-color-scheme: dark)").matches
  setIsDark(savedTheme === "dark" || (!savedTheme && prefersDark))
}, [])
```
